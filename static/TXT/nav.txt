Изменение фона меню
Чтобы изменить фон навигационной панели (navbar), вы можете переопределить класс bg-primary или просто задать свой стиль:

<style>
  .navbar {
    background-color: #2c3e50 !important;
  }

  .navbar .nav-link {
    font-weight: 600;
    color: #ecf0f1 !important;
  }

  .navbar-brand {
    font-weight: 700;
    color: #ecf0f1 !important;
  }
</style>
